# Package structure and rationale 

This repo separates:

1) **paper-ready outputs** (`paper/`)
   What a journal editor/reviewer needs: figures, source tables, compact reports.

2) **reproducible plotting packages** (`parts/`)
   Each part can be run independently and has its own README.

3) **core pipeline code** (`src/`)
   Included for completeness, but not required to reproduce the plotted figures from canonical tables.
